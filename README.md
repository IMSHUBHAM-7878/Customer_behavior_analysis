# Customer_behavior_analysis
Customer behavior analysis project using Python , SQL and data visualization tools .
📊 Customer Behavior Data Analytics Project
📌 Overview

This project focuses on analyzing customer behavior data to generate actionable business insights. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard creation in Power BI, and report generation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

📂 Dataset

The dataset contains customer transaction and behavioral information such as:

1.Customer ID

2.Purchase history

3.Product details

4.Transaction dates

5.Sales amount

6.Payment method

7.Region / Location

The dataset was loaded and processed using Python before being stored and queried in SQL databases.

🛠 Tools & Technologies Used

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL / MySQL / SQL Server)

Power BI (Dashboard & Data Visualization)

Jupyter Notebook

Gamma (Report & Presentation Creation)
